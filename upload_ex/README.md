# 파일 업로드와 다운로드
블로그
- [파일 업로드와 다운로드](https://jddng.tistory.com/294)

간단한 파일 업로드와 업로드한 파일을 조회 및 다운로드 예제 구현

## 구현 내용

- 파일 업로드를 위한 클래스 구현
  - ItemRepository : Item 관리
  - UploadFile : 실제 파일 명과 서버에서 관리하는 파일 명 관리
  - FileStore : 파일 관리


- 파일 업로드 구현

- 업로드한 파일 다운로드
  - 이미지 다운로드 요청 처리

- 업로드한 이미지 파일 웹페이지에 보이도록 구현
  - 이미지 요청 처리
